# Escolha de Carreira na Tecnologia: Backend

Projeto desenvolvido para praticar conceitos básicos de desenvolvimento web utilizando HTML e CSS.

## Sobre o Projeto

Esta página apresenta uma breve explicação sobre a área de desenvolvimento Backend, abordando seu papel no funcionamento de aplicações e sistemas.

O conteúdo foi desenvolvido como atividade prática para exercitar a criação de páginas web estáticas e a organização de conteúdo utilizando HTML e CSS.

## Tecnologias Utilizadas

* HTML5
* CSS3

## Funcionalidades

* Exibição de conteúdo informativo
* Inserção de imagens
* Estruturação de página web
* Estilização com CSS

## Objetivos de Aprendizagem

Durante o desenvolvimento deste projeto foram praticados:

* Estruturação de páginas HTML
* Organização de conteúdo
* Utilização de imagens
* Estilização com CSS
* Conceitos básicos de desenvolvimento web
